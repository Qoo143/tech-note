# Array 相關的操作方法

這邊放置陣列操作

### push、pop、shift、unshift

::: details 案例示範

1. push( )

```jsx
const arr = [1, 2, 3];
arr.push("5");
console.log(arr); //[1, 2, 3, 5]
```

2. pop( )

```jsx
const arr = [1, 2, 3];
arr.pop();
console.log(arr); //[1, 2]
```

:::
